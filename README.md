# Tomas Misrahi - Growth & Performance Marketing Portfolio

Metodología Data-Driven | Analítica Avanzada | Background Full-Stack

## Sobre el Proyecto
Este repositorio aloja el código fuente de mi portfolio profesional. Fue desarrollado desde cero (zero-based) para materializar y certificar mi enfoque tanto analítico-tecnológico como de Growth Marketing. Actúa como una prueba técnica en vivo sobre mis capacidades de Optimización de la Tasa de Conversión (CRO), arquitectura web sustentable, accesibilidad y velocidad extrema de carga, sin templates pre-armados.

## Stack Tecnológico
- Lenguajes: HTML5, **Tailwind CSS 3**
- Lógica: Vanilla **JavaScript (ES6+)**
- Backend (Forms & CI/CD): **Netlify**
- Versionado: **Git & GitHub**
