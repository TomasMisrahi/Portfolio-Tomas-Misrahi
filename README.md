# Tomas Misrahi - Growth & Performance Marketing Portfolio

Metodología Data-Driven | Analítica Avanzada | Background Full-Stack

## Sobre el Proyecto
Este repositorio aloja el código fuente de mi portfolio profesional bilingüe (ES/EN). Fue desarrollado desde cero para materializar y certificar mi enfoque tanto analítico-tecnológico como de Growth Marketing. Actúa como una prueba técnica en vivo sobre mis capacidades de Optimización de la Tasa de Conversión (CRO), arquitectura web sustentable, accesibilidad y velocidad extrema de carga, asegurando un rendimiento óptimo sin depender de templates pre-armados.

**Características Principales:**
- **Soporte Bilingüe:** Versiones integras en Español e Inglés nativo, con transiciones impecables conservando la posición de lectura.
- **Diseño 100% Responsivo:** Interfaz adaptable "mobile-first" que garantiza legibilidad extrema, desde un iPhone antiguo de 320px hasta monitores *ultrawide*.
- **Desempeño Extremo y Clean Code:** Código minificado sin librerías pesadas (cero jQuery/React), optimizado para core web vitals y SEO.
- **Formularios Dinámicos con AJAX:** Integración asíncrona del lado del cliente para envíos directos a Netlify Forms sin molestas recargas de página.
- **Ventanas Modales UI/UX:** Sistema robusto implementado en JS puro, con efectos "Glassmorphism" integrados para la exhibición de Casos de Éxito.

## Stack Tecnológico
- **Estructura y Estilos:** HTML5, **Tailwind CSS 3** (Custom Classes & Glassmorphism)
- **Lógica e Interactividad:** Vanilla **JavaScript (ES6+)** (Smooth Scrolling, Form AJAX Data, DOM Lifecycle Mode)
- **Backend (Forms & CI/CD):** **Netlify**
- **Versionado:** **Git & GitHub**
